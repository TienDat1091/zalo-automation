Cần cấu hình Google Cloud Credentials

1. Vào Google Cloud Console
2. Tạo project mới hoặc chọn project có sẵn
3. Bật Gmail API tại APIs & Services > Enable APIs
4. Vào APIs & Services > Credentials
5. Tạo OAuth 2.0 Client ID (chọn Web application)
6. Thêm http://localhost:3000/api/email/auth/google/callback vào Authorized redirect URIs
7. Download file JSON và tải lên đây
